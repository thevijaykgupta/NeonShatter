# Neon Shatter

Classic Brick Breaker game with neon visuals.

## How to Run

**Windows:**
```
run.bat
```

**Linux/Mac:**
```bash
chmod +x run.sh
./run.sh
```

**Manual:**
```bash
javac -source 8 -target 8 -d bin -sourcepath src src/*.java
java -cp bin NeonShatter
```

## Controls

- Arrow Keys: Move paddle
- ENTER/SPACE: Start game

## Features

- 4 different brick layouts (Grid, Wave, Circle, ZigZag)
- Neon glow effects
- Particle effects
- Score system

## Requirements

Java 8 or higher

## Play Online

The game is also available as a web version. Enable GitHub Pages in repository settings to play in browser.
